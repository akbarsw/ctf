# Mini Tebak Gambar

Mini web challenge sederhana untuk latihan CTF pemula.

## Cara main

1. Buka `index.html` di browser.
2. Perhatikan gambar yang muncul.
3. Masukkan jawaban dengan huruf kecil.
4. Jika benar, flag akan muncul.

## Deploy dengan GitHub Pages

Masuk ke Settings > Pages, lalu pilih:

- Source: Deploy from a branch
- Branch: main
- Folder: /root

Setelah itu GitHub akan memberikan link web challenge.
